# CrossStitchStash

Cross-platform macOS, iOS, and iPadOS app to organize your cross-stitch patterns and threads.

# Why?

I like cross-stitching, but my stash of threads and patterns has hit the point where I have trouble keeping track of the amount of thread I already have versus the amount I need to buy for an upcoming project. I also like Swift and developing iOS apps, but I don't have much experience with making a cross-platform app, nor do I have much experience with SwiftData or UI tests, so this is an excuse to learn more about those.

## Isn't it limiting if it's only for one type of craft?

If you want to use it for, say, embroidery, go ahead! In fact, I'd love to hear about other feature requests/use cases you may have for this app.

# Features

## Current

- View, add, and edit some basic pattern and thread info

## Planned

- Settings screen for things like measurement units (yards or meters)
- Better color scheming/design
- [Under investigation] iCloud data syncing across devices
- [Under investigation] Store thread and pattern thumbnail images
